# 开局

## 地图生成

地图生成完全可以根据个人喜好，下面列举几个可以考虑的点：

1. 新手建议选择和平模式，关闭虫子主动进攻，先熟悉生产、电力及资源运输。
2. 矿区资源密度可以适当调大，甚至是拉满。
3. 由于悬崖会严重影响前期规划，且悬崖炸弹需要石油资源，建议关闭悬崖生成。
4. 资源点分布更远的地图更适合火车运输，想玩火车运输的小伙伴可以考虑这点。

## 基本按键操作

> 注：`LMB`和`RMB`分别指鼠标左右键。

**玩家与用户界面**

| 操作                     | 按键            |
| ------------------------ | --------------- |
| 玩家移动                 | `W`,`A`,`S`,`D` |
| 打开关闭背包             | `E`             |
| 取消手持物品             | `Q`             |
| 旋转目标                 | `R`             |
| 丢在地面上               | `Z`             |
| 捡起附近地上的物品       | `F`             |
| 细节显现（**建议开启**） | `Alt`           |
| 地图                     | `M`             |

**设施操作**

| 操作                       | 按键              |
| -------------------------- | ----------------- |
| 复制鼠标所指向设施         | `Q`               |
| 放置设施蓝图               | `Shift`+`LMB`     |
| 向/从设施放置/拾取全部物品 | `Ctrl`+`LMB`      |
| 自动放置设施               | 按住`LMB`，并移动 |

**战斗操作**

| 操作     | 按键            |
| -------- | --------------- |
| 玩家移动 | `W`,`A`,`S`,`D` |
| 射击目标 | `Space`         |
