# 开局

## 基本按键操作

> 注：`LMB`和`RMB`分别指鼠标左右键。

**玩家与用户界面**

| 操作                     | 按键            |
| ------------------------ | --------------- |
| 玩家移动                 | `W`,`A`,`S`,`D` |
| 打开关闭背包             | `E`             |
| 取消手持物品             | `Q`             |
| 旋转目标                 | `R`             |
| 丢在地面上               | `Z`             |
| 捡起附近地上的物品       | `F`             |
| 细节显现（**建议开启**） | `Alt`           |
| 地图                     | `M`             |

**设施操作**

| 操作                       | 按键              |
| -------------------------- | ----------------- |
| 复制鼠标所指向设施         | `Q`               |
| 放置设施蓝图               | `Shift`+`LMB`     |
| 向/从设施放置/拾取全部物品 | `Ctrl`+`LMB`      |
| 自动放置设施               | 按住`LMB`，并移动 |

**战斗操作**

| 操作     | 按键            |
| -------- | --------------- |
| 玩家移动 | `W`,`A`,`S`,`D` |
| 射击目标 | `Space`         |
